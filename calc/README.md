# Calculator ✨

Привет! Добро пожаловать в стековый калькулятор 🧮
Проект поддерживает:
- выполнение команд калькулятора,
- GUI на JavaFX,
- динамическую загрузку команд из JAR,
- сохранение/загрузку состояния,
- логирование через Log4j2.

---

## Что нужно для запуска 🚀

Перед стартом убедись, что установлены:
- **JDK 25** (в `pom.xml` задано `source/target = 25`),
- **Maven 3.9+**.

Проверка:

```bash
java -version
mvn -version
```

---

## Установка и сборка 🔧

В корне проекта выполни:

```bash
mvn clean package
```

Почему именно `package`:
- в проекте команды загружаются из JAR-файлов,
- они создаются на фазе package,
- пути к ним берутся из `src/main/java/me/ivy/calc/command-jars.conf`.

После команды выше должны появиться:
- `target/calc-1.0-basic_commands.jar`
- `target/calc-1.0-binary_commands.jar`

---

## Запуск приложения (GUI) 🌿

```bash
mvn javafx:run
```

Откроется окно калькулятора, где можно вводить команды, смотреть стек и переменные 🙂

## Запуск тестов ✅

```bash
mvn test
```

## Поддерживаемые команды

- `PUSH <value>` — положить число/переменную в стек
- `POP` — снять верхнее значение
- `DEFINE <name> <value>` — задать переменную
- `SQRT` — квадратный корень от верхнего значения
- `+`, `-`, `*`, `/` — бинарные операции над верхними элементами стека

Пример:

```text
DEFINE a 4
PUSH a
SQRT
```

---

## Логи 📝

Используется Log4j2 (конфиг: `src/main/resources/log4j2.xml`).

Логи пишутся:
- в **stdout**,
- в файл **`calculator.log`** (в корне проекта).

---

## Полезные заметки 💡

- Строки можно разделять `;` для нескольких команд в одной строке.
- Комментарии начинаются с `#`.
- При ошибках команда не «роняет» приложение — возвращается сообщение вида `ERR ...`.
